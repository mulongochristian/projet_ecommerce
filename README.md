## 📌 Project Overview

This project presents a complete **Big Data & Artificial Intelligence pipeline** for a European e-commerce marketplace.

The system processes multiple heterogeneous data sources, performs data cleaning and feature engineering, detects fraudulent reviews, classifies customer support tickets using NLP, and provides interactive business dashboards while ensuring GDPR compliance and data security.
